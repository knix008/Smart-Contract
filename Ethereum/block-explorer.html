How to Connect

MetaMask Setup

1. Open MetaMask

2. Click on network dropdown

3. Select "Add Network"

4. Enter the following details:

  • Network Name: Private Ethereum
  • RPC URL: http://localhost:32801
  • Chain ID: 3151908
  • Currency Symbol: ETH

Remix IDE

1. Go to remix.ethereum.org

2. Click on "Deploy & Run Transactions" tab

3. Select "Web3 Provider" as environment

4. Enter: http://localhost:32801

Test Network Connectivity

Command Line Testing

Use these curl commands to test the network from terminal:

Test Network ID:

curl -X POST -H "Content-Type: application/json" --data '{"jsonrpc":"2.0","method":"net_version","params":[],"id":1}' http://localhost:32801


Get Latest Block:

curl -X POST -H "Content-Type: application/json" --data '{"jsonrpc":"2.0","method":"eth_blockNumber","params":[],"id":1}' http://localhost:32801


Get Block Details:

curl -X POST -H "Content-Type: application/json" --data '{"jsonrpc":"2.0","method":"eth_getBlockByNumber","params":["latest",false],"id":1}' http://localhost:32801
